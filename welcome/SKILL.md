# Welcome

이 스킬은 저장소의 기본 구조와 작성 방식을 설명합니다.

## 목적

- `SKILL.md` 형식의 최소 예시를 제공합니다.
- 새 스킬을 만들 때 복사 기준점으로 사용합니다.

## 작성 규칙

- 설명은 짧고 구체적으로 적습니다.
- 필요한 경우 `scripts/`, `references/` 폴더를 같은 디렉터리에 둡니다.
- 자동 실행이 필요한 스크립트는 꼭 검토 가능한 형태로 둡니다.
